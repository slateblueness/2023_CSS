transtition: 전환 효과

전환을 사용할 때는 전환되기 전 디자인과 전환된 후 디자인, 총 두 가지가 필요하다
전환되기 위한 조건(hover, checked 등)이 충족되면 전환되는 모습을 보여 준다
차후에는 JS를 이용해서도 전환 효과를 줄 수 있다

전환할 때, 여러 속성 이름으로 전환 효과를 줄 수 있다
전환을 추가할 때 ','를 통해 속성 이름과 시간을 추가
추가한 모든 값을 전환으로 사용할 때는 all 명령어를 사용하면 된다 ex) transtition: all 1s;

after를 이용해서 마우스를 올리면 배경색이 검정으로 바뀌고, 다른 글자가 나오게 전환

전환 효과를 주기 전에, 해당 태그에 전환했을 때 들어갈 디자인이 먼저 적용되는지를 확인한 후, 전환 효과를 주면 빠지는 내용 없이 넣기가 쉽다
선택자 after는 새로운 내용을 추가하는 선택자이다
때문에 after의 값과 :hover:after의 값을 따로 주어야 hover했을 때 전환이 가능하다
