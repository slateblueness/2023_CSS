크기

크기는 size 속성으로 변경할 수 있고, 기본 크기는 p태그의 글자 크기이다.
h1태그의 글자는 이미 200% 크기로 출력이 되며, 동일한 font-size(200%)를 넣어 주면 다른 글자들도 h1와 같은 크기가 된다.

안녕하세요

태그의 크기를 구하는 방법 height/width: 콘텐츠의 크기
padding: 패딩의 크기
border: 테두리의 크기
margin: 마진의 크기
콘텐츠 + 패딩 + 테두리 + 마진

box-size border-box 변경

box-size를 border-box로 변경하고 작성하면 height/width의 값이 border+padding+contents 값으로 바뀐다
크기값이 중요한 경우에는 border-box로 바꿔서 작성
border-box를 쓰는 쪽이 크기 계산이 편하기 때문이다

margin값-auto

margin 속성값 auto

margin 속성값 auto 값을 사용해 줄 때는 display가 block 속성이어야 하고, contents 크기가 부모 태그보다 작아야 한다.
(부모 크기와 같으면 마진이 들어갈 공간이 없기 때문)
margin-auto를 사용하면 부모의 크기 기준으로 중앙으로 정렬된다.
margin-auto는 div태그와 함께 자주 사용된다.

아웃라인 outline

아웃라인입니다

outline은 border와 동일하게 사용할 수 있다
그려지는 위치도 동일하게 그려진다
단, 두께는 크기의 범위에 들어가지 않기 때문에 계산할 필요가 없다

outline과 box-size에서 border-size로 작성했을 때,
*outline을 사용하면 border값 계산 X
*box-size를 사용하면 padding과 contents(height/width)를 따로 계산할 필요가 없음 -> contents값만 계산
>> 요소의 크기 계산: margin + contents
