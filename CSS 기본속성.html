CSS 기본속성

rgba(0, 0, 0, 0.5)는 네 번째 값은 alpha 값으로 0~1까지의 값을 가진다
0: 완전 투명, 1: 완전 불투명(투명도 없음)

background

background-color와 같이 background 내에서 값을 세분화해서 넣을 수도 있지만 background를 통해서 한 번에 값을 넣을 수도 있다
background-image를 이용해서 여러가지 이미지를 넣을 수 있고 필요하다면 그라디언트(그라데이션) 색상도 추가할 수 있다
attachment, position, repeat을 이용하여 이미지의 위치나 반복 여부를 변경할 수 있다
